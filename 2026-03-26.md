## 13:00 | Q: Internal Server Error — SQLite datetime 함수 MS-SQL 호환 문제
**질문:** 배포 후 Internal Server Error 발생
**실패 원인:** 코드 전체가 SQLite datetime() 함수로 작성되어 있어 MS-SQL에서 인식 못함
**해결:** database.py의 _translate_sql에 양방향 변환 추가
- datetime('now','+9 hours') → DATEADD(hour, 9, GETDATE())
- datetime(column,'+9 hours') → DATEADD(hour, 9, column)
- datetime('now') → GETDATE()
- INSERT OR IGNORE → INSERT
- kakao_members 참조도 booking_pages, admin_pages에서 제거
